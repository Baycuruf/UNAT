# Node.js Authentication API Boilerplate

Bu proje; güvenli, ölçeklenebilir ve üretime hazır bir Kimlik Doğrulama (Auth) altyapısı şablonudur.

## Özellikler

- **Kayıt Ol & Giriş Yap** (JWT - JSON Web Token)
- **Şifre Sıfırlama** (Email ile Token gönderimi)
- **Rol Bazlı Yetkilendirme** (Admin / User)
- **Güvenlik Önlemleri:**
  - Rate Limiting (Brute-force koruması)
  - Helmet (Header güvenliği)
  - NoSQL Injection Koruması
  - XSS Koruması
- **Password Hashing** (Bcrypt)
